# Java 8

The release that introduced functional-style programming to mainstream Java.

## Features

- [Lambdas](src/main/java/itb/java/examples/java8/Lambdas.java) — anonymous function syntax for instances of functional interfaces.
- [Streams](src/main/java/itb/java/examples/java8/Streams.java) — declarative pipelines for filtering, mapping, and reducing sequences.
- [Optional](src/main/java/itb/java/examples/java8/Optionals.java) — explicit, type-safe "value or absence" container.

Build & test:

```bash
../build.sh java8 test
```
