# Java 7

Highlights of the language and API changes introduced in Java SE 7:
<https://www.oracle.com/java/technologies/javase/jdk7-relnotes.html>

The demos here are written as plain classes with a `main()` method; matching JUnit 5 tests live alongside them under `src/test/java/itb/java/examples/java7/`. Sources still showcase Java 7-era syntactic features, though they're compiled to Java 8 bytecode because JDK 25's javac no longer supports targeting Java 7 directly.

## Features

- [Binary Literals](src/main/java/itb/java/examples/java7/BinaryLiterals.java)
- [Catching Multiple Exception Types and Rethrowing Exceptions with Improved Type Checking](src/main/java/itb/java/examples/java7/CatchMultiple.java)
- [Type Inference for Generic Instance Creation (Diamond Operator)](src/main/java/itb/java/examples/java7/CreationWithTheDiamondOperator.java)
- [Strings in switch Statements](src/main/java/itb/java/examples/java7/StringsInSwitchStatements.java)
- [The try-with-resources Statement](src/main/java/itb/java/examples/java7/TryWithResourcesStatement.java)
- [Underscores in Numeric Literals](src/main/java/itb/java/examples/java7/UnderscoresLiterals.java)
